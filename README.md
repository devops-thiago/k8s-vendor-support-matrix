- **DOKS**: DigitalOcean Kubernetes
- **OpenShift**: Red Hat OpenShift Container Platform
- **RKE**: Rancher Kubernetes Engine
- **RKE2**: Rancher Kubernetes Engine v2
- **TKG**: VMware Tanzu Kubernetes Grid
- **KOPS**: Kubernetes Operations
